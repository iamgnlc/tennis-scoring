# Tennis Scoring

React demo app to simulate tennis scoring algorithm.

## Install dependencies

```sh
$ yarn install
```

## Start app

```sh
$ yarn start
```

## Run test suite

```sh
$ yarn test
```

Unit test written using https://testing-library.com/.
